Home Server & Media Automation

Project Type: Personal Infrastructure

Tech Stack: Docker, Ubuntu Server, Jellyfin, Immich, qBittorrent, Radarr, Sonarr, Prowlarr, Bazarr, Tailscale, Portainer

Overview

I transformed an old desktop PC into a fully functional home server for self-hosting media and automating content downloads. The system is built with Docker and managed using Portainer for ease of control and monitoring.

Features

  • 📺 Jellyfin – Personal media streaming for movies and TV shows
  • 📷 Immich – Self-hosted Google Photos alternative for automatic photo backup
  • 🔁 Radarr / Sonarr / Prowlarr – Automated movie & TV series download pipelines
  • ⬇️ qBittorrent (headless) – Torrent management with web UI
  • 🌐 Tailscale – Remote secure access to the server via VPN
  • ⚙️ Portainer – Web-based Docker container management
